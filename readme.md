# Mintlify Landing Page (HTML + CSS Only)

Ye project **sirf HTML aur CSS** use karke banaya gaya hai (koi JavaScript ya frameworks nahi). Ye Mintlify ke landing page ka ek basic clone hai jo clean aur simple code follow karta hai.

---

## Features 

- **Structured Navbar**: Logo, links aur CTA buttons ke saath proper navigation bar.
- **Hero Section**: Announcement badge aur email input ke saath.
- **Logo Grid**: Companies ke logos ka clean layout.
- **Feature Cards**: GIFs aur images ke saath features dikhane wale cards.
- **Responsive Design**: Basic responsiveness taaki alag-alag screens par sahi dikhe.
- **Clean Code**: Easy-to-understand HTML structure aur standard CSS.

---

## Kaunsi technologies use hui hain

- **HTML5**: Page ke structure ke liye.
- **CSS3**: Styles, layout, aur formatting ke liye.
- **Design Assets**: SVG icons aur images (assets folder mein).




## Locally Kaise Run Karein 

### Option 1: Direct Open
1. Project ko download/clone karein.
2. `index.html` file ko apne browser (Chrome, Safari, etc.) mein open karein.

### Option 2: VS Code Live Server 
1. VS Code mein project folder open karein.
2. **Live Server** extension install karein.
3. `index.html` par right-click karein aur **Open with Live Server** select karein.
